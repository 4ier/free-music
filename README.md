# 自由的音乐

## 介绍
　　国内音乐版权开始完善，本项目收集来自互联网中的音乐资源。<br>
　　集合各家音乐的搜索结果，并实现试听功能。<br>
　　目前支持QQ，网易，酷狗音乐。<br>
　　同样支持mv的观看。
## 注意　　
　　禁止使用此工具进行下载。<br>
　　使用此工具获取的内容与本人无关。<br>
## 技术
　　javaweb框架使用SpringMVC和Spring。<br>
　　发送HTTP请求使用OkHttp。<br>
　　前端方面，ui使用的MD风格，改版后ui框架使用的是[MDUI框架](https://github.com/zdhxiong/mdui/)<br>
